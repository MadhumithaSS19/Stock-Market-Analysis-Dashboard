# Stock Market Analysis Dashboard

## 📌 Project Overview

The Stock Market Analysis Dashboard is an interactive Power BI report developed to analyze financial performance metrics such as revenue, profit, quantity, and discount trends. The dashboard provides insights into regional performance, monthly revenue patterns, category-wise profitability, and operational status tracking.

This project demonstrates how business intelligence tools can be used to monitor financial data and support strategic decision-making.

---

## 🎯 Objectives

- Analyze overall revenue and profit performance.
- Monitor quantity and discount metrics.
- Evaluate regional profit distribution.
- Track monthly revenue trends.
- Identify category-wise profit contributions.
- Monitor operational status performance.

## 📈 Key Insights

### Revenue Trends
- Revenue shows fluctuations across different months.
- Peak revenue was observed during the middle of the year.
- Monthly trend analysis helps identify seasonal patterns.

### Profit Analysis
- Total profit reached 2.89M.
- Regional profit contributions are distributed across East, West, North, and South regions.

### Category Performance
- Category-wise analysis highlights the most profitable segments.
- Profit contribution varies significantly between categories.

### Operational Performance
- Completed transactions account for the highest quantity.
- Status-based analysis helps monitor business efficiency.

### Business Value
- Provides a consolidated view of financial performance.
- Enables data-driven investment and operational decisions.
